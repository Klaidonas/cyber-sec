up::[[Asset Based Risk Assessment]]

- Vulnerability : Weakness that can be exploited
- Identified through
	- Vulnerability analysis
	- Audit reports
	- Penetration testing
	- Scanning tools
- Threat : Anything that can exploit a vulnerability
	- Hackers
	- Natural disasters
	- System failure
	- Accidental human interference
